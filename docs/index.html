3. 컨셉 — 모임 생물학

네이버 카페가 진화하지 못한 진짜 이유

네이버 카페·당근의 본질은 공간 제공자다. "방을 줄게, 알아서 채워라." 모임이 만들어진 뒤엔 관여 안 한다. 결과:

  • 카페 99%가 생성 → 며칠 활성 → 죽음 사이클로 끝남
  • 운영자는 왜 내 카페만 죽는지 모름
  • 가입자는 어느 카페가 진짜 살아있는지 모름 (회원 30만 명이지만 죽은 카페에 낚임)

진짜 막힌 지점: 네이버 카페는 카페의 생존에 무관심. 우리가 끼어들 자리.

30~40대 엄마(타깃 #1)의 숨은 결핍 5

  1. 사라진 1인칭 — "○○맘"으로만 호명됨
  2. 판단 검증 결핍 — 매일 수십 개 결정을 검증할 곳이 없음
  3. 시간 빈곤 + 인지 과부하 — 5분 단위 멀티태스킹
  4. 죄책감 화폐 — 모든 결정에 죄책감
  5. 수평 동료 부재 — 같은 처지 친구를 새로 사귀기 어려움

강팀이 도입한 신규 심리 레버 2개

A. 대리 결정 (Decision Outsourcing) — 내가 결정하기 무서운 걸 집단의 판단으로 옮긴다. 매일의 결정을 집단 응답과 함께 → 죄책감·실패 책임 분산.
B. 정체성 회복 (Identity Restoration) — "○○맘"이 아니라 결혼 전 직업·전공으로 호명되는 1분. 다른 어떤 앱도 못 주는 가치 → LTV 락인.

매일 들어오게 만드는 4중 조합

대리 결정(A) + 정체성 회복(B) + 호기심 갭(#11) + 인정(#2). 표준 12레버에 신규 2개를 더해 14레버 체계.

4. 헌법 6원칙

  1. 플랫폼은 게시판 위에 있다. 카페 안엔 게시판이 있고, 그 위에 심리 모듈을 카페가 켤 수 있다. 게시판 강화·검색·태그 단순 개선은 답이 아니다.
  2. 익명 + 정체성 회복은 카페별 옵션. 정체성 회복 모듈을 켠 카페만 결혼 전 직업·전공 호명. 기본은 닉네임. "○○맘" 호명은 전역 금지.
  3. 응답 전 결과 비공개 (커튼) — 결정 모듈에서만. 일반 게시판엔 해당 없음. 사회적 동조 편향 차단.
  4. 댓글권은 보상 — 심리 모듈에서. 공감 받은 한 줄 이유에만 댓글권. 일반 게시판은 자유 댓글.
  5. 수익은 광고가 아니다. 운영 SaaS / 정체성 프리미엄 / 데이터 자산 / 장례식 흡수 / 거버넌스 SaaS 5종.
  6. 영상은 폰에 산다. 셋로그 모듈의 영상은 폰에만 저장 + 메타만 서버. 사라짐이 기능 약속.

#게시판/모듈정의강한 모임
1오늘의 결정매일 1개 결정, 30초 응답, 24h 후 결과엄마·이직·운동·투자
2이름 회복실주 1회 30분 익명 라운지, 현재 정체성 언급 금지엄마·경단녀·은퇴자·동문
3내일의 나에게어제 후회한 사람 → 내일 같은 일 할 사람에게 메모이직·육아·등산·자취
4심박수 게시판모임 활동 실시간 그래프 + "심폐소생술 신청"모든 모임 (특히 대형)

단계핵심
1. 제안제목 + 요약 + 예상 효과 + 내가 맡을 부분(선택)
2. 양방향 투표찬성/반대 + 한 줄 이유 필수. 반대표가 더 가치 있음. 응답 전 결과 비공개
