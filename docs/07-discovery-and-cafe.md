### D01 결정 요약
- **상단**: 로고 + 검색·알림 아이콘 (검색은 *카페 내부 보조*가 메인이지만 디스커버리에도 진입점 유지)
- **오늘의 모임 결정** (Q1 b) — 매일 19시 풀스크린 모달 가능, 시간 외엔 카드 형태
- **내 모임 카루셀** — 사용자 명시 *카드 형식*. 가로 스크롤. *만든 모임 1번 자리 고정*
- **필터 칩 3개** (Q1과 무관, 사용자 명시) — 지역·인기·회원수. 1축 1선택. 다중축 X
- **모임 카드 A안** (사용자 명시 그대로) — 좌측 60px 로고, 우측 최근글 2 + 인기글 2 (위아래)
- **결핍 카드** — 빈 결과·검색 0건일 때 모임 생성 유도 (아뱅 Q2 안 1 적용)
- **FAB** — 모임 만들기 진입

### 카드 B안 (펼치기형, A안 답답할 시 백업)
한 줄로 표시하다 탭하면 확장. *UI 디자이너가 두 안 다 와이어 그려서 비교*.

## 3. 필터 UI — 칩 3개 상시

## 4. 모임 만들기 5단계 (M01~M05)

```
M01 카테고리 선택        → 모임 / 동호회 / 동네 / 회사 / 가족 / 기타
M02 세부 태그 입력       → 지역·연령·관심사 (사용자 자유 입력 + 자동 추천)
                          → 좁을수록 살아남기 쉬움 (강팀 알림)
M03 이름·설명·로고      → 로고는 이모지 + 자동 색상 추천 (디자인 부담 0)
M04 첫 게시판 선택      → 자유 게시판 1개 자동 생성 (기본)
                          → 추가 게시판은 *나중에* 운영자가 [+게시판]으로
M05 모듈 토글 (선택)    → 오늘의 결정 / 이름 회복실 / 내일의 나에게 / 심박수 4개
                          → 처음엔 다 OFF. 운영자가 14일 뒤 처방 받고 켤 수 있음
```

**4분 컷**. 마찰 0이 핵심.

│                                              │
│ ┌─[모듈] (켜진 것만) ──────────────────┐    │
│ │ 🎯 오늘의 결정 (24h 뒤 결과 받기)      │   │
│ │ 🌿 이름 회복실 (이번 주 토 21:00)       │   │
│ │ ✉ 내일의 나에게 (어제 47개 메모)       │   │
│ │ ♥ 심박 진단판 (운영자만)               │   │
│ └────────────────────────────────────────┘   │
│                                              │
│ [멤버] [정보] [활동로그] [내부 검색]         │
└─────────────────────────────────────────────┘

**결정**:
- 게시판이 위, 모듈이 아래 (게시판이 *입구*, 모듈이 *확장*)
- 운영자만 [+게시판] / [⚙] 보임
- *모임 활동 로그* (C06)는 *공개* — 운영자 권력 행사가 멤버에게 보임 (헌법 권력 견제)

## 6. 게시판 생성 플로우 (M10~M12)

```
M10 게시판 카탈로그       → 기본 (자유/공지/질문) + 특수 (아뱅 Q5의 7종)

                            게시판 담당 임명 (선택)
```

## 7. 권력 구조 화면 (M40~M42, B06, C06)

**M40 부운영자 초대**:
```
멤버 프로필 → [부운영자로 초대] →
   권한 프리셋 선택:
     [ ] 일반 부운영자 (기본 5종)
     [ ] 강퇴 전담 (강제 탈퇴만)
     [ ] 게시판 전담 (게시판 관리만)
     [ ] 커스텀 (5종 체크박스)
   → 초대 발송 → 멤버가 [수락/거절] → 활성
```

**B06 강제 탈퇴**:
```
멤버 메뉴 → [강제 탈퇴] →
   사유 1줄 필수 입력 (욕설·도배·다른 사유)
   → 즉시 추방 + 본인에게 사유 푸시
   → C06 활동 로그에 *공개 기록*
   → 14일 재가입 불가
```

## 8. 회원 관리 자동 규칙 (M50)

운영자가 모임별로 ON/OFF + 임계값 조정:

| 규칙 | 조건 (편집 가능) | 자동 처분 |
|---|---|---|
| 욕설 자동 옐로 | 키워드 사전 매칭 N회 | 옐로카드 7일 (봉인 두루마리 연출) |
| 도배 자동 옐로 | 5분 내 글 N개 작성 | 옐로카드 3일 |
| 광고 자동 옐로 | URL 포함 + 모임 외부 도메인 | 옐로카드 7일 |
| 신고 누적 자동 옐로 | 같은 글에 신고 5건+ | 옐로카드 14일 + 글 자동 숨김 |
| 3진 아웃 자동 탈퇴 | 옐로카드 누적 3개 | 즉시 탈퇴 + 14일 재가입 불가 |
| 회복의 길 | N일 무위반 시 | 카운트 1 차감 |
| 명예 회복 청원 | 멤버 30%+ 찬성 시 | 카운트 0 리셋 |

## 12. UI 디자이너에게 넘기는 모호점 (이번 라운드 6개)

1. **모임 카드 A안 vs B안** — 둘 다 와이어 → 사용자 픽
