# 심박 (Shimbak / Heartbeat)

> 누구나 카페를 만드는 *멀티 모임 플랫폼*. 네이버 카페가 "공간 임대업"이라면 **심박은 모임 생물학**이다.
> 카페의 *생사*를 책임지는 곳.

**시범 환경**: 강서구·3~7세 유아 엄마 모임을 *플랫폼 1호 모임(Genesis)*으로 보존 + 동시에 자유 모임 생성 허용
**구 이름**: 동네맘 (Dongne-Mom) — 멀티 카페 피벗으로 *심박*으로 개명

---

## 한 줄로 무엇이 다른가

기존 네이버 맘카페가 *"정보 게시판"*이라는 정체성에 갇혀 진화 못 했다면, 동네맘은 맘카페의 본질 — *"내 결정을 검증받고 싶은 익명의 약자"* — 을 직접 건드린다.

**핵심 두 신규 심리 레버**:
- **대리 결정 (Decision Outsourcing)** — 매일 1개의 결정을 100명과 동시에 내려서 책임을 분산
- **정체성 회복 (Identity Restoration)** — "○○맘"이 아닌 *결혼 전 이름·직업*으로 호명되는 1분

**수익**: 광고 아님. *결정 데이터의 B2B 리서치 자산화* (동네 가게가 의사결정 리포트를 산다).

## 플랫폼 구조 — 3계층

```
[플랫폼 레벨]
  ├ 디스커버리 홈 — 모임 카드 (좌 로고 + 우 최근글 2 + 인기글 2)
  └ 매일 19시 "오늘의 모임 결정" — 가입 결정 큐레이션 의식

[카페 레벨]                       (운영자가 만드는 단위)
  ├ 기본 게시판: 자유 / 공지 / 질문 — 공통 7+게시판별 3 기능
  ├ 옵션 모듈 (운영자 처방 받고 켬):
  │    ├ 오늘의 결정      (눈팅 90% → 응답자 60%)
  │    ├ 이름 회복실       (처음 보는 사이 → 동료)
  │    ├ 내일의 나에게     (콘텐츠 수명 7배)
  │    └ 심박수 진단판     (운영자 기본 ON)
  ├ 상시 가동 (기본 ON, 운영자가 끌 수 있음):
  │    └ 셋로그 동시 영상   (시간 윈도우 안에서만, 영상은 폰 저장)
  └ 아이디어 게시판 (4단계 거버넌스)

[운영자 권력]
  ├ Owner / Co-Owner / Board Manager / Member / Visitor 5계층
  ├ 강제 탈퇴 + 자동 규칙 5종 + 재미 요소 8개
  └ 견제 4종: 아이디어 자동 채택 / 공개 로그 / 고백 게시판 / 사망 경고
```

## 헌법 — 어기면 제품이 죽는 5원칙 (피벗 후 갱신)

1. **플랫폼은 게시판 *위에* 있다.** 카페 안엔 게시판이 있고, 그 위에 *심리 모듈*을 카페가 켤 수 있다. 플랫폼 자체는 *모임을 발견·생성·운영*하는 레이어.
2. **익명 + 정체성 회복은 카페별 옵션.** 정체성 회복 모듈을 켠 카페는 결혼 전 직업·전공으로 호명. 기본은 닉네임.
3. **응답 전 결과 비공개 (커튼) — *결정 모듈에서*.** 일반 게시판엔 해당 없음.
4. **댓글권은 보상 — *심리 모듈에서*.** 일반 게시판은 자유 댓글.
5. **수익은 광고가 아니다.** 운영 SaaS / 정체성 프리미엄 / 데이터 자산 / 장례식 흡수 / 거버넌스 SaaS 5종.

상세는 [`.claude/knowledge/momcafe/project-context.md`](.claude/knowledge/momcafe/project-context.md).
